API Endpoints

POST /task

Method: POST

Path: /task

Creates a new task.

Request Body

The request body should include the following key:

  • name: The name of the task
  • description: The description of the task
  • status: The status of the task

GET /tasks

Method: GET

Path: /tasks

Gets all record

GET /tasks/filter?status=all&sort=desc&limit=10&skip=0

Method: GET

Path: /tasks/filter?status=all&sort=desc&limit=10&skip=0

Gets record with pagination

GET /task/:id

Method: GET

Path: /task/:id

Returns requested record

PUT /task/:id

Method: PUT

Path: /task/:id

Returns the updated record

Request Body

The request body should include the following key:

  • name: The name of the task

DELETE /task/:id

Method: DELETE

Path: /task/:id

Returns deleted record
